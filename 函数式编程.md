1. 函数柯里化

   

   让柯里化函数自动判断参数的个数、实现多参数的调用方法并实现混用

   ```js
   const curryn = func => {
       return function curryFunc(...args) {
           if (args.length < func.length) {
               return (...params) => {
                   return curryFunc.apply(null, args.concat([...params]));
               }
           }
           return func.apply(null, args);
       }
   }
   ```
