# pnpm

1. #### workspace

   - 在根目录添加`pnpm-workspace.yaml`文件

     ```yaml
     packages:
       # all packages in direct subdirs of packages/
       - backend
       - frontend
       - packages/*
     ```

   - pnpm --stream -r 【command】
     - -stream 可以在终端上试试查看输出并运行指定的命令
     - -r  --recursive 在每个workspace下边的proj执行该命令
